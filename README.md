# Proyecto Intermodular - Gestión de Almacén
David Adrián Ciocanel Milea - 1ºK DAM
## Descripción
Proyecto diseñado para que una pequeña empresa pueda tener un registro fácil de los productos de su almacén
## Estructura del Proyecto
- database/: Scripts y herramientas para la gestión de la base de datos.
- docs/: Documentación del proyecto.
- `src/`: Código fuente de la aplicación.
Esto deberá ser un commit adicional al commit inicial, dale un mensaje semántico (docs).
